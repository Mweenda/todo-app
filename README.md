# todo-app
A simple todo app 
## Features

- Create, update, and delete tasks
- Responsive UI using Bootstrap
- Django-powered backend

## Tech Stack

- **Backend:** Django
- **Frontend:** HTML, Bootstrap (CSS & JS)
